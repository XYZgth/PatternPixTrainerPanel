\doxysubsection{Detailed Description}
Repräsentiert einen Datenpunkt für die Visualisierung gleitender Durchschnitte. 

Diese Klasse wird verwendet, um Kennzahlen wie durchschnittliche Zeit und durchschnittliche Fehler pro Trainingseinheit (Session) darzustellen -\/ Diagramme mit Syncfusion.

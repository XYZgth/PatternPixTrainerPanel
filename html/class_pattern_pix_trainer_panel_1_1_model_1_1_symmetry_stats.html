Detailed Description

Repräsentiert Daten zu einer bestimmten Symmetrie.

Diese Klasse enthält grundlegende Kennzahlen wie durchschnittliche Bearbeitungszeit und Fehleranzahl für eine Symmetrie-Kategorie.

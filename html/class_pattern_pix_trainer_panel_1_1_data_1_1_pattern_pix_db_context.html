Detailed Description

Datenbankkontext für die PatternPix-Anwendung.

Stellt den Zugriff auf die Datenbanktabellen Children und Trainings bereit und konfiguriert die Datenbankverbindung sowie die Beziehungen zwischen Entitäten.

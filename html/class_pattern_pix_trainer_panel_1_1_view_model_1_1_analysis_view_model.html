Detailed Description

ViewModel für die Analyse-Ansicht der PatternPix Trainer Anwendung.

Diese Klasse verwaltet die Datenanalyse und Statistiken für Trainingseinheiten, einschließlich Filterung, Peer-Vergleiche und Chart-Darstellung.

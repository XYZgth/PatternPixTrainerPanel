Detailed Description

Basisklasse für alle ViewModels im MVVM-Pattern.

Diese Klasse stellt die gemeinsame Funktionalität für ViewModels bereit, insbesondere Zugriff auf den EventAggregator von Prism für die Kommunikation zwischen Komponenten.
